# Snap-on

Snap-on Incorporated is a global manufacturer and marketer of tools, equipment, diagnostics, repair information, and systems solutions for professional users in vehicle repair, industrial, and other industries. The company provides diagnostic tools, electronic parts catalogs (EPC), repair information systems (ShopKey Pro, SureTrack), and shop management software integrating with dealer management systems (DMS) through its EPC integration program and strategic technology partnerships.

## APIs

- [Snap-on Electronic Parts Catalog (EPC)](https://sbs.snapon.com/automotive/epc/epc5/) - Parts lookup, pricing, and ordering with DMS integration for 125,000+ dealerships in 130+ countries.
- [ShopKey Pro Repair Information System](https://www.shopkeypro.com/) - Comprehensive automotive repair information including OEM data, wiring diagrams, TSBs, and SureTrack real-fix information.
- [SureTrack Expert Information](https://www.snapon.com/EN/US/Diagnostics/Information--Software-Products/SureTrack-Expert) - Cloud-based expert knowledge platform with verified Real Fixes from millions of successful repair orders.
- [Snap-on Diagnostics Software](https://www.snapon.com/EN/US/Diagnostics/Information--Software-Products/Diagnostic-Tool-Software) - Professional diagnostic software for 50+ vehicle makes with OEM gateway access.
